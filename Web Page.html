Introduction to Programming

Lesson 1 : Introduction to web page and HTML

The Web

The web is made of HTML pages called hypertext markup language. This is the basis for almost all webpages and the links which connect these pages are called hyperlinks.

Major pieces of the web

The major pieces of the web are

The Browser : Makes requests to the internet
The internet : Uses HTTP whch is the main protocol of the web to communicate with the browser and the server
The Server : These are computers which host web files.

HTML is the heart of the web which is made up of text content, markup, reference to other documents and links to other pages.

HTML markups are used to change a particular text into some form(for example : BOLD or Italics). These markups mainly use tags for this purpose.

Tags

Tags are used in HTML markups to perform a particular set of functions such as to change a particular text into bold or Italicized text, to refer to other documets such as to include images, to include links to other pages.

Tags consist of an opening tag, the contents and a closing tag. All these three together is called an element.

HTML Attributes are also used along with tags. These Attributes have a name and a value which are used to provide links to other pages in HTML.
For example, Anchor tags have href as their attribute. Also, a tag can have multiple attributes.

Inline and Block

Tags like the p tag and the br tag can be classified into Inline and Block. As we can see, both the tags work in a similar fashion. But, they do have a significant difference between them. The br tag just breaks the line whereas the p tag creates an invisible box around the content where it is used. Also, the br tag is a void tag and does not have a closing tag.

Some of the other tags that were introduces were span and div tags which are inline and block tags respectively.

Lesson 2: Creating a structured Document

Introduction

HTML pages are building blocks of a web page. It is analogous to constructing a house.
They form the structures of the house as to which should be the living room and which room should be the bed room.

The CSS are for styling like what colour should be the walls etc.

Tree like structures

The HTML pages are made of branches of structures.
Analysing structures and styling in a given web page is done using the "Developer tools" in the web. Other elements can exist inside of the HTML elements and this can be compared to a family tree.

This structure, in a browser is shown as a series of nested boxes which contains elements within elements.

Indented HTML and Boxes

A HTML document is deeply indented and can be seen as a wave on the left side of the document. When there is a lot of indentations seen it means that those many elements are nested in the document.

Programming in Text Editors

Text Editors such as Sublime Text and Codepen.io is used by programmers to write their code. This makes it easier for them to used tags. Some text editors use the ending tag automatically when a tag is introduced in the code.

HTML, CSS and DOM

The HTML is a language used to write the web page which has a syntax and there are some rules associated as to how to write text on the web.
CSS is used for styling and it determies how the web page looks.
Document object model abbreviated as DOM is for communicating within elements. Tags are helpful in turning elements in trees.

Lesson 3 : Adding CSS style and Structure

Avoiding Repetition and Errors

Copying and pasting some attributes of styling in HTML elements and rewriting the same code may lead to unnecessary repetition of code writing.It also causes errors while programming and hence has to be avoided.

While designing a web page, the HTML can be written as a structure within divs and classes can be assigned to each of these divs. The styling can be applied by mentioning only the class name in the CSS file. This helps the programmers to avoid repetition of codes.

CSS

Cascading styling sheets is shortly referred to as CSS. As the name suggests, in CSS, the styling is appied to the document in an inherited styling method where the style assigned for a parent content is inherited by the tag that encloses the content.

This is done by giving a class name to each div in the HTML document.
